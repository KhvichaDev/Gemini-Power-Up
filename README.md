# 🚀 Power-Up for Gemini — The Ultimate Productivity Suite for Google Gemini

> **Supercharge Gemini with 25+ tools:** Task Manager, Code Editor, AI Debate, Translator, Image Editor, Notes, Timer & more.

Transform Google Gemini from a simple chatbot into a full-featured professional workspace. Power-Up for Gemini adds 25+ powerful tools directly into the Gemini interface — no separate apps, no context switching, everything right where you work. 

👉 **[Add to Chrome now and unlock the full potential of Gemini!](https://chromewebstore.google.com/detail/power-up-for-gemini/cnhmdgnlbdjngjpcbkahodfgccggonhp)**

### 📋 PROJECT & TASK MANAGEMENT
A complete project management system built right into Gemini. Create projects, organize tasks with priorities and due dates, add subtasks, and track progress — all without leaving your AI workspace.

* **Task Manager** — Full-featured with projects, priorities (Critical/High/Medium/Low), subtasks, due dates, and completion tracking
* **Plan with AI** — Describe your idea and let AI generate a complete project plan with tasks, subtasks, and deadlines automatically
* **Daily Briefing** — Start every day with an intelligent summary of your pending tasks, upcoming deadlines, and overdue items
* **Pomodoro Timer** — Built-in focus timer with customizable work/break sessions, sound alerts, and cross-tab synchronization

---

### 💻 CODE EDITOR & PATCHER
A professional-grade code editing environment that transforms how developers use Gemini.

* **Code Editor** — Syntax highlighting for 50+ languages powered by CodeMirror, with a dedicated editing workspace
* **Smart Patcher** — Automatically detects code changes from AI responses and applies them to your files with conflict resolution
* **Version Control** — Built-in versioned backups with intelligent diff comparison so you never lose a single change
* **Chat Integration** — Send your code to Gemini with full AI context for intelligent refactoring and debugging
* **GitHub Integration** — Sync project snapshots and version history directly to your GitHub repositories
* **WordPress Sync** — Auto-deploy patched code directly to your WordPress site
* **NotebookLM Integration** — Export code to NotebookLM with security rules that filter sensitive files and credentials

---

### 🤖 AI DEBATE ENGINE
The feature no other extension offers. Create AI-vs-AI debates to explore topics from multiple perspectives.

* **Multi-Agent Debates** — Pit multiple AI models against each other (Gemini, GPT) with custom roles and system prompts
* **Custom Agents** — Define each agent's personality, expertise, and argumentation style
* **Debate from History** Launch debates based on existing conversations for deeper analysis
* **Pause & Resume** — Full control over the debate flow with real-time status logging

### 🎨 IMAGE EDITOR
Edit AI-generated images without leaving Gemini. A full canvas-based editor with professional tools.

* **Crop & Resize** — Precise cropping with aspect ratio lock and custom dimensions
* **Text Overlay** — Add styled text with custom fonts, colors, sizes, and alignment
* **Shapes & Drawing** — Freehand brush, rectangles, polygons, and shape masks
* **Filters** — Brightness, contrast, saturation, grayscale, sepia, hue-rotate, blur, and invert
* **Transform** — Rotate and flip with full undo/redo history
* **Zoom & Pan** — Detailed editing with mouse wheel zoom and keyboard shortcuts
